<h2>📌 Project Overview</h2>

<p>
Shree Sai Restaurant is a front-end restaurant ordering system designed to simulate
a digital dining experience inside a restaurant.

Customers sitting at a table can:
</p>

<ul>
<li>Select their table number</li>
<li>Browse Veg / Non-Veg menu</li>
<li>Add items to the bill</li>
<li>View live price calculation</li>
<li>Place an order</li>
<li>Cancel items within 5 minutes</li>
</ul>

<p>
The project focuses on building a <b>simple yet interactive restaurant UI</b> using pure frontend technologies.
</p>

---

<h2>🎯 Features</h2>

<ul>
<li>🍽️ Restaurant digital menu system</li>
<li>🥦 Veg and 🍗 Non-Veg toggle menu</li>
<li>📋 Organized categories (Starters, Main Course, Roti, Drinks)</li>
<li>🧾 Real-time bill generation</li>
<li>➕ Add food items instantly</li>
<li>➖ Remove items from bill</li>
<li>⏱️ Order cancellation allowed only for 5 minutes</li>
<li>📊 Automatic total calculation</li>
<li>📱 Responsive grid layout</li>
<li>✨ Smooth UI animations</li>
</ul>
